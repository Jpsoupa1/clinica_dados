# Clínica Dados

Este projeto simula a base de dados de uma clínica médica, com funcionalidades para cadastrar pacientes, registrar atendimentos e manipular informações clínicas. O sistema é desenvolvido em Python, com foco em aplicar estruturas de dados e lógica de programação.

## Funcionalidades

-  Cadastro de pacientes com informações básicas (nome, CPF, idade, etc.)
-  Registro de consultas médicas
-  Armazenamento e recuperação de dados
-  Consulta de histórico de atendimentos
-  Listagem de pacientes cadastradose
